### EWN Job Task Frontend

### Live Link:

```
https://ewn-frontend.vercel.app/
```

### Technologies

- Next.js(app router)
- TypeScript
- TailwindCSS
- Shadcn/ui
- OTP implementation
- React-Redux
- Redux-toolkit
- RTK Query
- React-hook-form
- Framer-motion
- Js-cookie

## How to run the project

- Clone the main branch of the repo
- Run "npm i" for installation
- Run "npm run dev" to show the ui in the browser
